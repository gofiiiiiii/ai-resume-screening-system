# AI Resume Screening System

This project analyzes resumes and compares them with job descriptions u give using NLP techniques and give the match in percentage.

## Features
- Upload resume (PDF)
- Extract text
- Calculate match score
- Simple UI using Flask

## Tech Stack
- Python
- Flask
- Scikit-learn
- NLP

## How to Run
```bash
pip install -r requirements.txt
python app.py
